# Purpose

Calculate two-body correlation function from XDATCAR files. Only work
for a cubic system.


<a id="org14deabc"></a>

# Compile

Try "make cleanall" and "make"


<a id="org82300ac"></a>

# Usage

cat XDATCAR | pdfxdat<sub>cubic</sub>


<a id="org9041e05"></a>

# Output

A pdf file for unary and multiple pdf files for alloys.
